# atm系统
##系统介绍
 该系统是一个小课设，比较简单，主要实现了插卡登录取框退卡打印凭证等功能，由于该课设主要重点设计模式的使用，所以功能相对较少，使用javafx作为前端界面，
 也没有采用数据库，在登录时默认账号为1，密码为1，账号里面有10000元
